# Carpeta `research`

Este directorio contiene documentación de investigación, diseño y arquitectura para el proyecto de recreación de un sistema tipo Reactable.

## Contenido

- `reactable_overview.md`: Notas generales sobre el proyecto original Reactable (historia, conceptos, capacidades).
- `reactivision_technical_notes.md`: Notas técnicas sobre reacTIVision y el tracking de marcadores fiduciales.
- `reactable_requirements_and_use_cases.md`: Requisitos funcionales/no funcionales y casos de uso inspirados en Reactable.
- `architecture_concepts.md`: Boceto de arquitectura y componentes principales para nuestra versión moderna multiplataforma.

Estos documentos son de trabajo, están pensados como apuntes vivos que se irán refinando durante el proyecto.
